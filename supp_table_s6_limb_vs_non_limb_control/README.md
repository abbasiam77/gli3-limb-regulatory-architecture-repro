# Supplementary Table S6: limb versus validated non-limb Gli3-region enhancer comparison

This module contains Supplementary Table S6 materials for the revised Gli3 limb regulatory landscape manuscript.

Supplementary Table S6 compares 13 curated Gli3 limb enhancers with 41 validated non-limb VISTA enhancers from the broader Gli3 region.

- S6a provides enhancer-level signal values.
- S6b provides class-level summary statistics.

The analysis includes combined HOX13 signal, active enhancer score, GLI3 signal, PITX1 average signal, and related chromatin/transcription-factor layers used to assess overlap and heterogeneity between validated enhancer classes.
