# Supplementary Figure S3: limb versus non-limb Gli3-region enhancer comparison

This module contains the final Supplementary Figure S3 PDF and supporting files for the revised Gli3 limb regulatory landscape manuscript.

Supplementary Figure S3 summarizes signal distributions across 13 curated Gli3 limb enhancers and 41 validated non-limb VISTA enhancers from the broader Gli3 region. The figure visualizes combined HOX13 signal, active enhancer score, GLI3 signal, and PITX1 average signal.

The purpose of this analysis is to show overlap and heterogeneity between validated limb and non-limb enhancer classes, supporting cautious multi-layer interpretation of bulk chromatin and transcription-factor datasets.
