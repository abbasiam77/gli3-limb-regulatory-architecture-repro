# Main Fig. 1 (mm9) reproducibility module

## Purpose
This module contains reproducibility assets for manuscript **Main Fig. 1**, focused on the **mm9-based integrative 3D regulatory architecture context** around the *Gli3* locus in mouse hindlimb.

## Contents
- `config/` — figure-specific configuration (e.g., plotting/track settings)
- `docs/` — source notes, IGV session, provenance-related supporting files
- `figures/` — exported figure/panel assets
- `scripts/` — scripts used for processing and figure assembly support

## Notes
This module has been reorganized for reviewer-friendly navigation as part of repository-wide cleanup.
Genome-build handling and manuscript-wide provenance are documented at the repository level.
